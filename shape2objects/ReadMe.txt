シェイプキーを別々のオブジェクトに分解するスクリプトです。

詳しい説明は省きますが、下記のパラメータを変更すると作成されるオブジェクトの間隔を変更出来ます。

offsetX = 3
offsetY = 0
offsetZ = 0
dstX = 0
dstY = 0
dstZ = 0

下記のパラメータはシェイプキーを一番最初に作成したときのデフォルトのシェイプの名前です(英語UIの場合)。
名前が異なる場合には合うように変更してください。

nmBasis = 'Basis'

例)新規データを日本語にしている場合

nmBasis = '基本ノイズ'
